# Huffman compression & decompression 

## How to use 
Compile it with : **g++ -Wall Huffman.cpp main.cpp -o (your .exe name)** 

## Run 
The program needs 2 arguments to run, in that order :  
*text to encode* *frequences file*

## Frequences file
